Description

Soulworker Patcher was made to easily patch latest translations to your Soulworker Client without much risk of corrupting your original files when updating.

It also can download latest translations from the SWHQ translation team and apply them at any time.

Functions

  • Backups the original files and replaces them with the downloaded translations.
  • Auto-updates the Soulworker client when it's outdated.
  • Unicode codepage fix to support other unicode languages (example: Thai, Korean...)
  • Direct login for faster startup.

Common Problems

  • "Access denied" error

    Follow this guide to give your User read and write permissions to the soulworker folder.

  • "Application cannot be started. Contact the application vendor." error on startup

    Uninstall the Patcher (not Soulworker!) and install it again with the installer provided in the link above

  • At server selection screen, the server name looks like some gibberish...

    It's a side effect of the unicode codepage fix. It will not affect anything else.
